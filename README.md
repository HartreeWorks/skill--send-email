# Send Email

A Claude Code skill for sending emails via Gmail SMTP with preview and confirmation. Uses Gmail App Passwords for authentication - no Google Cloud project or API keys required.

## Documentation

See [skill.md](./skill.md) for complete documentation and usage instructions.

## Installation

```bash
npx skills add HartreeWorks/skill--send-email
```

Or install all HartreeWorks skills: `npx skills add HartreeWorks/skills`
